LXDMXWiFi_Library

LXDMXWiFi_Library encapsulates functionality for sending and receiving DMX over and ESP8266 WiFi connection.
LXDMXWiFi is a virtual class with concrete subclasses LXWiFiArtNet and LXWiFiSACN which specifically
implement the Artistic Licence Art-Net and PLASA sACN 1.31 protocols.
