Introduction

LXDMXWiFi encapsulates functionality for sending and receiving DMX over WiFi. It is a virtual class with concrete subclasses LXWiFiArtNet and LXWiFiSACN which specifically implement the Artistic Licence Art-Net and PLASA sACN 1.31 protocols.

Note: LXDMXWiFi_library requires WiFiUdp included with the ESP8266 Arduino package.

Note: For sending packets larger than 512 bytes, ESP8266 WiFi Library v2.1 is required.

setUniverse

set universe for sending and receiving 

public

virtual void setUniverse (
    uint8_t u );
Parameters
u

universe 0/1-255

Discussion

First universe is zero for Art-Net and one for sACN E1.31. Additionally, for Art-Net high nibble is subnet, low nibble is universe. For example, setUniverse(0x12) with Art-Net is subnet 1, universe 2, the same as setSubnetUniverse(1,2). With sACN, setUniverse(0x12) is universe 18.

universe

universe for sending and receiving dmx 

public

virtual uint8_t universe (
    void );
Return Value

universe 0/1-255

Discussion

First universe is zero for Art-Net and one for sACN E1.31. Art-Net divides the universe byte with the high nibble equal to the subnet, and the low nibble representing the universe.
