Introduction

LXWiFiSACN partially implements E1.31, lightweight streaming protocol for transport of DMX512 using ACN http://tsp.plasa.org/tsp/documents/published_docs.php

LXWiFiSACN is primarily a node implementation. It supports output of a single universe of DMX data from the network. It does not support merge and will only accept

packets from the first source from which it receives an E1.31 DMX packet.
